## Overview

This is an example document rendered with the **AdAdvisor brand template**. It uses the same logo, colors, fonts, and page furniture as the contractor agreement.

> Use this template for offer letters, briefs, internal memos, or any deliverable that should look like an official Ad Advisor AI document.

## Sections

- Markdown headings (`##`) become section headers with a magenta left bar.
- Bullets, **bold**, *italic*, [links](https://adadvisor.ai), and `inline code` all work.
- Tables are styled with a dark navy header row and zebra striping.

| Field | Value |
|---|---|
| Brand color | `#d946ef` magenta |
| Accent color | `#f97316` orange |
| Font | Helvetica Neue |

### Custom Callouts

You can drop raw HTML into the markdown for branded components:

<div class="callout">
  <strong>Pro tip:</strong> Use a callout box to highlight an important takeaway.
</div>
