## Overview

This is an example document rendered with the **Joud brand template**. It uses the deep-maroon palette, rose accent, Inter body type, and the running page furniture from the Joud landing-page design system.

> Use this template for technical primers, internal briefs, customer reports, or any deliverable that should look like an official Joud AI document.

## Sections

- Markdown headings (`##`) become section headers with a rose left bar.
- Bullets, **bold**, *italic*, [links](https://joud.ai), and `inline code` all work.
- Tables are styled with a deep-maroon header row and zebra striping.

| Field | Value |
|---|---|
| Brand primary | `#B0506A` rose |
| Brand accent | `#3D1828` deep maroon |
| Body font | Inter |
| Code font | JetBrains Mono |

### Custom Callouts

You can drop raw HTML into the markdown for branded components:

<div class="callout">
  <strong>Pro tip:</strong> Use a callout box to highlight an important takeaway.
</div>
